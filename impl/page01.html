Головна сторінка Сторінка з теоріює Приклади розв'язання задач Сторінка з контрольною роботою

Відомості для опрацювання теми:
"Неінерціальні системи відліку. Сила інерції"

Анотація:
В ході вивчення тем, заявлених у шапці документа,  від учінів вимагається вислухати загальні положення теми "Неінерціальні системи відліку",  вивчити поняття "Сила інерції", "Відцентрова сила інерції", "Сила Каріоліса" та "Штучне тяжіння".  В ході навчнання, разом з учнями, роз'язати задачи для тренування та обговорити отримані результати.
Форма контролю для перевірки знань учнів - контрольна робота.
План опрацювання:
  1. Головна сторінка:
    • Анотація;
    • Відомості про автора;
  2. Теоретичні відомості:
    • Неінерціальні системи відліку;
    • Сила інерції;
    • Відцентрова сила інерції;
    • Сила Коріоліса;
    • Штучне тяжіння;
  3. Розв'язок задач:
    • Задача 01. Знайти час, за який тіло спуститься по похилій площині.
    • Задача 02. Знайти прискорення точок осі цилінда.
  4. Контрольна робота;
